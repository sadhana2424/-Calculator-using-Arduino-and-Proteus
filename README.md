## 🧮 Calculator using Arduino and Proteus

## 📌 Project Overview
The Calculator using Arduino and Proteus is a simulation-based embedded systems project that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. The project is designed using Arduino IDE and simulated in Proteus Design Suite, allowing users to perform calculations without requiring physical hardware.

## 🎯 Objective
Perform basic arithmetic calculations.
Learn Arduino programming using simulation.
Understand keypad interfacing with Arduino.
Display calculation results on an LCD.
Develop embedded systems programming skills.

## 🛠️ Software Required
Arduino IDE
Proteus Design Suite

## ⚙️ Working Principle
The user enters two numbers using the 4×4 keypad.
An arithmetic operator (+, −, ×, ÷) is selected.
The Arduino processes the entered values.
The result is calculated instantly.
The final output is displayed on the 16×2 LCD in the Proteus simulation.

## ▶️ How to Run
Open the project in Arduino IDE.
Compile the code and generate the HEX file.
Open the Proteus simulation.
Load the HEX file into the Arduino Uno.
Run the simulation.
Enter numbers and operators using the virtual keypad.
View the calculated result on the LCD.
